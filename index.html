Pertanyaan Umum

Apakah ada biaya bulanan atau sewa hosting?
Untuk Paket Express, tidak ada biaya bulanan sama sekali untuk server/hosting. Kami menggunakan teknologi static modern dan GitHub Pages sehingga web Anda online gratis selamanya. Anda hanya membayar sekali di awal.
Bagaimana jika saya tidak mengerti coding?
Website ini memang dirancang khusus untuk orang awam. Anda tidak perlu menyentuh kode sama sekali. Untuk mengubah harga produk, menambah stok, atau mengubah teks, Anda cukup mengetiknya di aplikasi Google Sheets (seperti Excel) dari HP Anda.
Berapa lama proses pembuatan website?
Untuk Paket Express, website bisa langsung jadi dan online dalam waktu 1x24 jam setelah Anda mengirimkan data lengkap (Logo & Produk). Untuk Paket Custom, proses memakan waktu 3-5 hari kerja tergantung tingkat kerumitan.
Apakah saya bisa menggunakan domain sendiri (.com / .id)?
Sangat bisa! Namun, harga paket belum termasuk biaya pembelian domain. Kami bisa membantu Anda menyambungkan website ini ke domain .com atau .id milik Anda sendiri.
